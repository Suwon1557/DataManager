- 일단 push는 당연히 feature/ui쪽에 해놓음. 만일 develop에 합쳐야한다면, 연락주면 하겠음.
- 아래의 세부 설명에서, '~하였음' 이라고 하면 그건 이미 구현 된 기능임.
- 기능 개발에 필요하다면 당연히 UI 일부 수정 가능. 기능/데이터 담당이 직접 해도 되고, UI담당에게 얘기하면 직접 하겠음.
- 모든 chart는 꺾은선 그래프여야하고, x축은 사진 번호여야함. 

## 전체 화면 구조 및 로직
- tcMain : 탭 구현.
    - 1. tcDataManager : 데이터 관리(탭1)
    - 2. tcTrainingTest : 학습/테스트(탭2)
- lblTitle : 타이틀(Data Manager) 써놓은 label.

### 데이터 관리(탭1)
- gbDataLoad : '데이터 불러오기' GroupBox
    - btnSelectFolder : 데이터 폴더를 불러오기 위한 Button. 해당 버튼 누를 시 Folder Dialog가 뜨고, 거기서 폴더 선택 시 txtFolderPath에 경로까지 뜨게 만들었음.

    - txtFolderPath : 선택한 폴더 경로 표시할 TextBox

    - btnCheckDataIntegrity : 데이터 무결성 검사 실행 버튼. 해당 버튼을 클릭할 시, 특정 조건의 데이터가 발견될 시 사용자에게 MessageBox로 알려줘야함
        - 특정 조건 : index 번호 중복, catalog에는 있으나 이미지 파일이 없음, image는 있으나 angle/throttle 값이 없음
    
- gbDataContent : '데이터 탐색' GroupBox
    - pbDataPreview : 현재 프레임 이미지 표시하는 PictureBox
    - dgvDataInfo : 현재 프레임의 데이터(데이터 수, 현재 이미지 번호, 조향값, 속도값) 표시하는 DataGriv
        - colDataName : 데이터 항목명 컬럼
        - colDataValue : 데이터 값 컬럼
    - lvDataItems : 프레임 목록 표시 및 선택할 수 있는 ListView

    - btnFilter : 버튼 클릭 시 조건에 맞는 데이터를 필터링해서 삭제하는 Button
        - 조건 : 조향값 == 0 혹은 속도값 == 0
    - btnDeleteRange : lvDataItems에서 선택한 데이터 or btnSetRange로 선택한 범위의 데이터를 삭제하는 Button
    - btnCancelDelete : 최근 삭제 취소(복구). 버튼 클릭 시 이전에 삭제한 데이터가 복구되어야함. 

    - btnPlay : 클릭시 이미지가 정방향으로 자동 재생하는 Button
    - btnStop : 클릭시 재생 정지하는 Button
    - btnReverse : 클릭시 이미지가 역재생하는 Button

    - tbPlaybackSpeed : 재생 배속 조절하는 TrackBar. 범위는 0.25 - 2까지 구현해놨음. 해당 컨트롤은 소수를 저장하지 못하여 정수로 25 - 200까지 저장하는 로직이니 참고. 
    - lblPlaybackSpeed : 현재 배속을 표시하는 Label. 현재 배속 표시 기능 이미 구현했음. 

    - tbImageNavigator : 프레임 이동시키는 TrackBar
    - btnSetRange : 삭제 범위 지정 시작. 해당 버튼을 누르고 TrackBar 조정시 주황식 마커(Panel로 구현됨)이 찍힘. 최대 2개의 마커가 찍히며 그 사이의 데이터가 삭제되는 기능이 필요함.
    - btnCancelRange : 범위 지정을 중간에 취소하는 Button(사용자가 잘못 눌렀을 시를 대비)
    - pnlImageRangeMarker : 범위 지정 시 TrackBar에 표시되는 마커로서의 기능 수행해야하며, 이미 구현 완료.

    - chtSteeringValue : 이미지 번호에 따른 조향값 보여주는 Chart
    - chtSpeedValue : 이미지 번호에 따른 속도값 보여주는 Chart
    
### 학습/테스트(탭2)

- gbTraningSetup : '데이터 학습' GroupBox
    - btnTrain : 학습 실행시키는 Button. Python을 실행시켜서 'Process.Start("python", "manage.py train");' 이 명령어를 통해 학습 시작. 
    - txtTrainingLog : 학습 로그 표시. 

- gbModelTest : '모델 테스트' GroupBox
    - pbTestPreview : 테스트에 사용될 이미지 순서대로 표시하는 PictureBox
    - btnStartTest : 모델테스트 실행할 Button. 이 버튼을 누르면 화면에 파란색 막대기(예측 조향값)과 초록색 막대기(실제 조향값)이 떠야함.

    - chtTestSteeringValue : 실제/예측 조향값 비교 chart. x축은 사진 번호.
    - chtTestSpeedValue : 실제/예측 속도값 비교 chart. x축은 사진 번호.
