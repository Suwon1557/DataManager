# 🚗 Data Manager 개발 계획서

**Data Manager**는 WinForms 기반으로 자율주행 데이터를 효율적으로 관리하고 AI 학습 과정을 시각화하기 위한 데이터 관리 애플리케이션 프로젝트입니다.

## 🛠 기술 스택
* **Framework:** .NET WinForms
* **Language:** C#, Python (데이터 학습 연동)
* **Data Format:** JSON (Donkeycar Catalog)

---

## ✨ 주요 기능

### 1. 데이터 탐색 및 관리
* **데이터 불러오기:** 폴더 선택을 통해 데이터를 로드하고 경로를 확인합니다.
* **데이터 무결성 검사:** 중복 인덱스, 파일 유실, JSON 오류 등을 학습 전에 미리 검사하고 로그를 출력합니다.
* **이미지 제어:** 프레임 이동 및 리스트 선택 기능을 지원하며, 자동 재생, 중지, 역재생, 배속(1배속, 2배속) 조절이 가능합니다.
* **AI 예측 시각화:** 이미지 위에 실제값과 예측값을 나타내는 막대기를 표시하여 차이를 직관적으로 비교합니다.

### 2. 데이터 편집 및 복구
* **필터링 및 삭제:** 특정 데이터를 필터링하거나 삭제할 수 있으며, 시작값과 끝값을 설정하여 여러 데이터를 일괄 삭제하는 기능을 지원합니다.
* **데이터 복구:** 백업 시스템을 통해 실수로 삭제한 최근 이미지 및 데이터를 복구할 수 있습니다.
* **급격한 변화 강조:** Angle 또는 Throttle 값이 급격하게 변하는 구간을 리스트에서 빨간색으로 표시하여 관리 효율을 높입니다.

### 3. AI 학습 및 통계
* **학습 실행:** Python 연동을 통해 학습을 수행하며 실시간으로 Epoch 및 Loss를 확인합니다.
* **데이터 그래프:** Angle 및 Throttle의 변화량을 시각적인 그래프로 제공합니다.
* **모델 테스트:** 예측값(Predicted)과 실제값(Actual)을 겹친 꺾은선 그래프로 시각화하여 모델의 정확도를 평가합니다.

## 🖼 UI 구성
본 프로그램은 단일 폼 내에서 데이터 전처리부터 모델 검증까지 통합된 워크플로우를 제공합니다.
* **데이터 탐색 영역:** 이미지 미리보기, 프레임 정보 및 재생 컨트롤러.
* **데이터 편집 영역:** 필터링, 범위 삭제 및 복구 버튼.
* **학습 및 통계 영역:** 학습 제어 인터페이스 및 데이터 분석 그래프.
